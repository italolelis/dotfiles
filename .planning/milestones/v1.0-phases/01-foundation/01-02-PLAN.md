<task type="auto">
  <name>Task 1: Rewrite install.sh as idempotent Stow-based installer</name>
  <files>install.sh</files>
  <action>
    Replace the entire contents of install.sh with a Stow-based installer. Use the skeleton from RESEARCH.md as a starting point but implement fully.

    **Shebang and safety:**
    ```bash
    #!/usr/bin/env bash
    set -euo pipefail
    ```
    Do NOT use `#!/bin/sh` — BASH_SOURCE and arrays require bash.

    **Constants:**
    ```bash
    DOTFILES="$HOME/.dotfiles"
    PACKAGES=(zsh git tmux starship ghostty ssh misc)
    ```
    Use a fixed `DOTFILES` constant — do NOT derive from BASH_SOURCE (see RESEARCH.md Pitfall 7).

    **Helper functions:**
    - `log()`, `ok()`, `info()`, `fail()` — simple formatted output with status indicators
    - Step-by-step progress: "Stowing zsh... done" per CONTEXT.md decision

    **Flow (in order):**

    1. **Banner** — Print "Dotfiles installer" header

    2. **Confirmation prompt** — Ask "Continue? (y/N)" unless `--force` or `-f` flag passed. The `--force` flag is specifically for devcontainer/automation use per CONTEXT.md.

    3. **install_homebrew()** — Auto-install Homebrew if `command -v brew` fails:
       ```bash
       /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
       ```
       Handle both Apple Silicon (`/opt/homebrew/bin/brew shellenv`) and Intel (`/usr/local/bin/brew shellenv`) paths.

    4. **require_stow()** — Verify GNU Stow is installed. If not, attempt `brew install stow` first. Then verify with `stow --version`. Print version for debug.

    5. **run_brew_bundle()** — Run `brew bundle install --file="$DOTFILES/Brewfile" --no-lock` (only if Brewfile exists — skip gracefully if missing, since Brewfile is created in plan 01-03).

    6. **backup_and_remove_conflicts()** — For each package, find files that would conflict with stow (regular files in $HOME at the same paths). Move them to `$HOME/.backup/dotfiles_$(date +%Y%m%d_%H%M%S)/`. This is the pre-stow conflict resolution per RESEARCH.md Pattern 3.
       - Use `find "$DOTFILES/$pkg" -type f` to list package files
       - For each, compute target path in $HOME
       - If target exists AND is NOT a symlink, move to backup dir
       - Skip if target is already a symlink (already stowed)

    7. **stow_packages()** — For each package in PACKAGES array:
       ```bash
       stow --restow --no-folding --target="$HOME" --dir="$DOTFILES" "$pkg"
       ```
       Use `--restow` (not `--stow`) for idempotency per RESEARCH.md Pattern 2.
       Use `--no-folding` to prevent directory folding per RESEARCH.md Pattern 4.
       Print "Stowing {pkg}... done" for each.

    8. **Done message** — "Done! Restart your shell or run: source ~/.zshrc"

    **Anti-patterns to avoid (per RESEARCH.md):**
    - Do NOT use `stow --adopt` — ever
    - Do NOT tap deprecated Homebrew taps (homebrew/bundle, homebrew/cask, etc.)
    - Do NOT run `brew upgrade` unconditionally
    - Do NOT use `stow --stow` on repeated runs — use `--restow`
    - Do NOT use `--dotfiles` flag — the package layout uses explicit path mirroring

    **Idempotency requirements (FOUND-03):**
    - `stow --restow` is idempotent by design (unstow then stow)
    - `brew bundle install` is idempotent (no-op for installed packages)
    - Backup function skips symlinks (already stowed = no action)
    - Homebrew install checks `command -v brew` first
