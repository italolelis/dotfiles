This is a full rewrite because the manual color scheme (30+ lines) is being replaced entirely by the catppuccin/tmux plugin, and major new sections are added (prefix, TPM, plugins, session persistence). The good settings from the existing config (base-index, splits, mouse, focus-events) are preserved.

New content for `tmux/.tmux.conf`:

```tmux
# ── Prefix ────────────────────────────────────────────────────────────────────
unbind C-b
set -g prefix C-a
bind C-a send-prefix

# ── General ───────────────────────────────────────────────────────────────────
set -g base-index 1
set -g pane-base-index 1
setw -g pane-base-index 1
set -g renumber-windows on
set -s escape-time 0
set -g history-limit 50000
set -g display-time 4000
set -g status-interval 5
set -g focus-events on
set -g mouse on

# ── Terminal colors ───────────────────────────────────────────────────────────
# tmux-256color for true color and italics support (replaces xterm-256color)
set -g default-terminal "tmux-256color"
set -as terminal-overrides ",*-256color:Tc"

# ── Splits and navigation ────────────────────────────────────────────────────
# Keep current path on split
bind | split-window -h -c "#{pane_current_path}"
bind - split-window -v -c "#{pane_current_path}"
unbind '"'
unbind %

# Vim-style pane navigation
bind h select-pane -L
bind j select-pane -D
bind k select-pane -U
bind l select-pane -R

# Reload config
bind r source-file ~/.tmux.conf \; display "Config reloaded"

# ── Catppuccin Mocha ──────────────────────────────────────────────────────────
set -g @catppuccin_flavor "mocha"
set -g @catppuccin_window_status_style "rounded"

# ── Plugins ───────────────────────────────────────────────────────────────────
# Order matters: catppuccin BEFORE resurrect+continuum (Pitfall 1 from RESEARCH.md)
# catppuccin rewrites status-right; if it runs after continuum, autosave hook disappears
set -g @plugin 'tmux-plugins/tpm'
set -g @plugin 'tmux-plugins/tmux-sensible'
set -g @plugin 'catppuccin/tmux#v2.1.3'
set -g @plugin 'tmux-plugins/tmux-resurrect'
set -g @plugin 'tmux-plugins/tmux-continuum'

# ── Session persistence ──────────────────────────────────────────────────────
set -g @continuum-restore 'on'

# ── Status bar (catppuccin modules) ──────────────────────────────────────────
set -g status-left-length 100
set -g status-right-length 100
set -g status-left ""
set -g status-right "#{E:@catppuccin_status_session} #{E:@catppuccin_status_uptime}"

# ── Auto-install TPM if missing ──────────────────────────────────────────────
if "test ! -d ~/.tmux/plugins/tpm" \
  "run 'git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm && ~/.tmux/plugins/tpm/bin/install_plugins'"

# ── Initialize TPM (must be last line) ───────────────────────────────────────
run '~/.tmux/plugins/tpm/tpm'
```

Key design decisions and what changed vs the old config:

1. **Prefix**: Added `Ctrl+a` (screen-style, per user decision). Old config used default `Ctrl+b`.
2. **Terminal**: `tmux-256color` replaces `xterm-256color`. True color override uses wildcard `*-256color:Tc` for broad compatibility.
3. **New general settings**: `escape-time 0` (no vim delay), `history-limit 50000`, `renumber-windows on`, `display-time 4000`.
4. **Preserved from old config**: `base-index 1`, `pane-base-index 1`, `| and -` split bindings with `#{pane_current_path}`, `unbind " and %`, `mouse on`, `focus-events on`.
5. **Enhanced**: Reload bind now shows `"Config reloaded"` message.
6. **New**: Vim-style pane navigation (`prefix + hjkl`).
7. **Catppuccin Mocha**: `@catppuccin_flavor "mocha"` with `rounded` window status style.
8. **Plugin order is critical**: `catppuccin/tmux#v2.1.3` is declared BEFORE `tmux-resurrect` and `tmux-continuum`. This prevents catppuccin from overwriting continuum's autosave hook on `status-right` (Pitfall 1 from RESEARCH.md).
9. **Session persistence**: `@continuum-restore 'on'` auto-restores the last saved session on tmux server start. tmux-continuum auto-saves every 15 minutes.
10. **TPM self-bootstrap**: If `~/.tmux/plugins/tpm` does not exist, tmux auto-clones it and runs `install_plugins`. This works on first run without any manual setup.
11. **TPM init is the last line**: Required by TPM — all `@plugin` declarations must precede it.
12. **Removed**: All manual color/design lines (30+ lines of red/yellow/black manual theme). Replaced entirely by catppuccin/tmux plugin.

**After applying this config:**
- Existing tmux sessions should be killed and restarted: `tmux kill-server && tmux`
- On first start, TPM will auto-clone and install plugins (takes a few seconds)
- Alternatively, start tmux and press `prefix + I` (Ctrl+a then Shift+I) to install plugins manually
