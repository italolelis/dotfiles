<task type="auto">
  <name>Task 1: Audit current machine state and generate Brewfile comparison</name>
  <files>Brewfile</files>
  <action>
    Generate a comprehensive view of what's currently installed vs. what's in the old Brewfile.

    **Step 1: Dump current machine state:**
    ```bash
    brew bundle dump --describe --file=/tmp/Brewfile.current --force
    ```
    This generates a Brewfile from everything currently installed.

    **Step 2: Read the existing Brewfile:**
    Read `install/Brewfile` to understand what was previously declared.

    **Step 3: Build comparison report for user:**
    Create a clear category-by-category comparison:

    For each category (CLI Tools, Development, Apps/Casks, Fonts, Taps):
    - **In old Brewfile AND installed:** Keep (recommend)
    - **In old Brewfile but NOT installed:** Drop (recommend) — list these explicitly since they represent stale entries
    - **Installed but NOT in old Brewfile:** Present to user for decision — these are "discovered" packages
    - **Deprecated entries:** Flag for removal (exa -> eza, deprecated taps)

    **Known deprecated items (from RESEARCH.md):**
    - `tap "homebrew/bundle"` — deprecated, brew bundle is built-in
    - `tap "homebrew/cask"` — deprecated
    - `tap "homebrew/cask-fonts"` — deprecated, fonts in main cask
    - `tap "homebrew/cask-versions"` — deprecated
    - `tap "homebrew/core"` — deprecated
    - `brew "exa"` — abandoned, community moved to `eza`
    - `brew 'mac-mouse-fix'` — this is a cask, not a formula

    **Mandatory inclusions (regardless of current install state):**
    - `brew "stow"` — required for this dotfiles setup
    - `brew "fzf"` — needed for Phase 2 shell integration

    **Mandatory exclusions (per CONTEXT.md decisions):**
    - No `mas` entries — Skip Mac App Store apps

    Present the comparison to the user and ask for approval/modifications per category.
  </action>
  <verify>
    <automated>test -f /tmp/Brewfile.current && echo "Machine state dumped" && cat /tmp/Brewfile.current | wc -l | xargs -I{} echo "{} lines in current state"</automated>
  </verify>
  <done>Current machine state dumped. Comparison report generated showing: packages to keep, packages to drop, new discoveries, and deprecated entries.</done>
</task>

<task type="auto">
  <name>Task 3: Write final Brewfile and remove old install/Brewfile</name>
  <files>Brewfile</files>
  <action>
    Based on user's approval from Task 2, write the final Brewfile.

    **Location:** `~/.dotfiles/Brewfile` (top-level, per RESEARCH.md recommendation — NOT in install/ subdir)

    **Format (per CONTEXT.md):** Organize into sections with comment headers:
    ```ruby
    # Taps
    # (only non-deprecated, non-default taps)

    # CLI Tools
    brew "coreutils"
    brew "stow"          # Required for dotfiles management
    brew "wget"
    brew "jq"
    # ... etc

    # Shell
    brew "fzf"
    brew "starship"
    # ... etc

    # Development
    brew "go"
    brew "node"
    # ... etc

    # Apps
    cask "ghostty"
    cask "rectangle"
    # ... etc

    # Fonts
    cask "font-jetbrains-mono"
    # ... etc
    ```

    **After writing Brewfile:**
    1. Remove old `install/Brewfile` — `git rm install/Brewfile`
    2. Remove empty `install/` directory if nothing else is in it — `rmdir install/ 2>/dev/null`
    3. Verify: `brew bundle check --file=Brewfile` — should pass (since packages are already installed)

    **Anti-patterns:**
    - Do NOT include deprecated taps
    - Do NOT include `mas` entries
    - Do NOT use `--lock` (Brewfile.lock.json adds noise)
  </action>

    <automated>cd /Users/italovietro/.dotfiles && test -f Brewfile && echo "Brewfile exists at root" && ! test -f install/Brewfile && echo "Old Brewfile removed" && grep -q 'brew "stow"' Brewfile && echo "stow is in Brewfile" && ! grep -q "homebrew/bundle" Brewfile && echo "No deprecated taps" && brew bundle check --file=Brewfile 2>&1 | head -3</automated>
  </verify>
  <done>Brewfile at repo root with user-approved packages organized into sections. Old install/Brewfile removed. No deprecated taps. stow included. brew bundle check passes.</done>
