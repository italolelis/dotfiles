New content for `ghostty/.config/ghostty/config`:

```
# Ghostty Terminal Configuration
# See: https://ghostty.org/docs/config for all options

# ── Font ──────────────────────────────────────────────────────────────────────
# JetBrains Mono — installed via Brewfile (font-jetbrains-mono cask)
# Ghostty has built-in Nerd Font glyph rendering; no patched variant needed
font-family = JetBrains Mono
font-size = 14
font-feature = +liga
font-feature = +clig

# ── Theme ─────────────────────────────────────────────────────────────────────
# Catppuccin Mocha is bundled into Ghostty — no separate file install needed
theme = Catppuccin Mocha

# ── Window ────────────────────────────────────────────────────────────────────
window-padding-x = 8
window-padding-y = 8
window-decoration = true
macos-titlebar-style = tabs
background-opacity = 0.95

# ── Cursor ────────────────────────────────────────────────────────────────────
cursor-style = block
cursor-style-blink = false

# ── Shell integration ─────────────────────────────────────────────────────────
shell-integration = zsh

# ── Clipboard ─────────────────────────────────────────────────────────────────
copy-on-select = true

# ── Quick terminal ────────────────────────────────────────────────────────────
# Global hotkey dropdown terminal — works system-wide even when Ghostty unfocused
# Requires macOS Accessibility permissions (System Settings → Privacy & Security → Accessibility)
quick-terminal-position = top
quick-terminal-screen = main
quick-terminal-autohide = true
quick-terminal-animation-duration = 0.2
keybind = global:cmd+grave_accent=toggle_quick_terminal

# ── Splits and tabs ──────────────────────────────────────────────────────────
# Supplementary to tmux — tmux is the primary split/pane manager
keybind = cmd+shift+d=new_split:right
keybind = cmd+d=new_split:down
keybind = cmd+shift+h=goto_split:left
keybind = cmd+shift+l=goto_split:right
keybind = cmd+shift+k=goto_split:up
keybind = cmd+shift+j=goto_split:down
```

Changes from existing config:
1. **Theme**: `"Monokai Pro"` → `Catppuccin Mocha` (unquoted Title Case — canonical since Ghostty 1.2.0+)
2. **Font**: Removed quotes from `font-family` (canonical Ghostty style); size 13 → 14; moved ligatures next to font section
3. **Quick terminal**: New section — `global:cmd+grave_accent` toggles dropdown terminal from top of screen
4. **Splits**: New section — cmd+shift+d/cmd+d for splits, cmd+shift+hjkl for navigation (vim-style)
5. **Organization**: Sections with comment headers for readability; comments explain Nerd Font glyph rendering, Accessibility requirement
6. **Preserved**: All existing settings (padding, decoration, tabs, opacity, cursor, shell integration, copy-on-select) are kept

**Important note for verification:** The quick terminal global keybind requires macOS Accessibility permissions. After applying this config and restarting Ghostty, the user must grant Ghostty access in System Settings → Privacy & Security → Accessibility for the hotkey to work.
